01 — Methodology

System architecture

A sequential pipeline from pair selection through signal generation, regime filtering, and portfolio accounting. Each stage is independently testable and documented across 25 versions.

pair selection
Engle-Granger cointegration on all ~45,150 pairs. ADF on OLS residuals, both directions, p < 0.05. Half-life filter 4–25 days. PCA strips 5 systematic factors (~58% variance); idiosyncratic residual test adds quality bonus. CUSUM + Hurst pre-filters. Quarterly re-selection.
spread construction
1-D Kalman filter estimates time-varying hedge ratio β_t. Spread = log P₁ − β_t · log P₂. Beta-weighted P&L: Δlog P₁ − β_t · Δlog P₂, allocation split total/(1+|β|) per leg.
signal rule
Rolling z-score with half-life-adaptive lookback. Entry |z| > 1.8, exit |z| < 0.5. Dynamic max hold = clamp(2.5 × half-life, 10–25 days).
regime gate
VIX bands: >30 → 0.5×, >40 → 0.25×. 63-day cumulative sector dispersion gate ("walking on ice": low VIX + high dispersion → 0.4×). Hard skip: >20% of trailing 63 days reduced-scale.
validation
19 walk-forward windows, 252-day train / 63-day test, rolling by one test window. Agent retrained from scratch each window. No data leakage between train and test periods.

03 — Controlled ablation

ML contribution: measured at +0.02pp/qtr

A transformer-based signal-quality scorer was trained on entry-outcome labels from each training slice and used for opportunity ranking. A controlled ablation — identical data, identical code, one environment flag — quantifies its contribution precisely. The BCE loss converges to the base-rate entropy in every run: 82–91% of |z| > 1.8 entries revert regardless of features, meaning the entry rule has already extracted the learnable signal.

04 — Disclosure

Honest assessment

Negative results are reported in full. Every bug, fix, and failed hypothesis is documented across 25 versions in the engineering log.

What the data shows
The system generates positive gross OOS returns (+1.30%/qtr, 17/19 windows profitable). It does not generate a deployable net edge. All five institutional cost profiles — quant HF (~5–7× leverage), pod shop (~4×), fundamental L/S (~1.5–2×), unlevered institutional, and retail — produce negative net returns on the five-year backtest. IS→OOS degradation of 75% is substantially above the 30–50% range typical of well-validated strategies. Whether this reflects regime break, overfit, or both is not settled by this work. The ML component was built, trained, and measured: its contribution is +0.02pp/qtr.
